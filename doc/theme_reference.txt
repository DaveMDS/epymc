
                EpyMC THEMES REFERENCE


Main elementary layout group: "emc/main/layout"
All the icons in icons.edc must be implemented

### Backdrop stuff ###
   * Required parts:
      "bg.swallow.backdrop1" -> edje SWALLOW
      "bg.swallow.backdrop2" -> edje SWALLOW  __unused__


### TopBar stuff ###
   * Required parts:
      "topbar.title" -> edje text
      "topbar.box" -> edje box

   * Signals received from the application:
      "topbar,show" "emc"
      "topbar,hide" "emc"


### MainMenu stuff ###
   * Required parts:
      "mainmenu.list" -> elm_list

   * Signals received from the application:
      "mainmenu,show" "emc"
      "mainmenu,hide" "emc"


### Browser: List View ###
   * Required parts:
      "browser.list.genlist1" -> elm_genlist style: "browser"
      "browser.list.genlist2" -> elm_genlist style: "browser"
      "browser.list.poster" -> edje swallow -> EmcRemoteImage
      "browser.list.info" -> edje TEXTBLOCK
      "browser.list.total" -> edje TEXT

   * Signals received from the application:
      "browser,list,show" "emc"
      "browser,list,hide" "emc"
      "browser_list_flip_left" "emc"
      "browser_list_flip_right" "emc"


### Browser: Grid View ###
   * Required parts:
      "browser.grid.gengrid" -> edje swallow
      "browser.grid.title" -> edje text part

   * Signals received from the application:
      "browser,grid,show" "emc"
      "browser,grid,hide" "emc"


### Video Player ###

### Dialog (panel) ###

   * Group: "emc/dialog/panel"

   * Required parts:
      "emc.text.title" -> edje text
      "emc.swallow.content" -> edje swallow
      "emc.box.buttons" -> edje box

   * Signals received from the application:
      "emc,dialog,show"
      "emc,dialog,hide"
      "emc,dialog,title,show" "emc"
      "emc,dialog,title,hide" "emc"
      "emc,dialog,buttons,show" "emc"
      "emc,dialog,buttons,hide" "emc"

   * Signal that should/must be sent to the app
      "emc,dialog,close" ""
      "emc,dialog,show,done" ""
      "emc,dialog,hide,done" ""
