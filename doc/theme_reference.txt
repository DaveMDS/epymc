
                EpyMC THEMES REFERENCE

All the icons and images in icons.edc must be implemented.

### Theme info ################################################################
   * Required data items:
      item: "theme.name" "The theme name";
      item: "theme.version" "0.1";
      item: "theme.author" "author name";
      item: "theme.info" "A more long description of the theme";
      item: "theme.generation" "XX";


### Main Layout ###############################################################
   * Group: "emc/main/layout"

   * Messagges:
      id: 12  TYPE: MSG_STRING  ->  MESSAGE_CLOCK_TIME (current formatted time)
      id: 13  TYPE: MSG_STRING  ->  MESSAGE_CLOCK_DATE (current formatted date)

### Video Player ###
   * Required parts:
      "videoplayer.video" -> edje SWALLOW
      "videoplayer.events" -> edje part with mouse_events = 1
      "videoplayer.subs" -> TEXTBLOCK (style: "epymc_subtitles_style")
      "videoplayer.controls.btn_box" -> edje BOX (will contain play,stop,etc buttons)
      "videoplayer.controls.btn_box2" -> edje BOX (will contain Video,Audio,etc buttons)
      "videoplayer.controls.slider" -> edje GROUP -> "emc/slider/default"
      "videoplayer.controls.position" -> edje TEXT
      "videoplayer.controls.length" -> edje TEXT
      "videoplayer.controls.poster" -> edje SWALLOW
      "videoplayer.controls.title" -> edje TEXT

      "videoplayer,hide" "emc"
      "videoplayer,controls,show" "emc"
      "videoplayer,controls,hide" "emc"

      "emc,dialog,title,hide" "emc"
      "emc,dialog,buttons,show" "emc"
      "emc,dialog,buttons,hide" "emc"

   * Signal that should/must be sent to the app:

      "emc,dialog,show,done" ""
      "emc,dialog,hide,done" ""
